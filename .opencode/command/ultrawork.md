### Phase 3: Aggressive TODO Tracking
- Create atomic, verifiable TODO items
- Mark `in_progress` before starting each
- Mark `completed` immediately after
- Never batch completions
- Track delegated work as separate TODOs

### Phase 4: Relentless Completion
- Continue until ALL TODOs are complete
- Verify every result before marking done
- Handle failures with retry or escalation
- Cancel all background tasks at completion
